Parameters

artistUri
Type: SpotifyWebApi.Model.UriSpotifyUri
The SpotifyUri for the artist.
albumTypes (Optional)
Type: SpotifyWebApi.Model.EnumAlbumType
Optional. A list of AlbumTypes that will be used to filter the response. If not supplied, all album types will be returned. Note multiple types can be selected like so: 'AlbumType.Album | AlbumType.AppearsOn'.
market (Optional)
Type: SystemString
Optional. An ISO 3166-1 alpha-2 country code. Provide this parameter if you want to apply Track Relinking.
limit (Optional)
Type: SystemInt32
Optional. The number of album objects to return. Default: 20. Minimum: 1. Maximum: 50.
offset (Optional)
Type: SystemInt32
Optional. The index of the first album to return. Default: 0 (i.e., the first album).

Type: PagingSimpleAlbum
The artists albums.
